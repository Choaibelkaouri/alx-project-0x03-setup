# alx-project-0x02-setup

هذا المستودع يحتوي على مشروع **Next.js + TypeScript + Tailwind CSS** المطلوب في مشروع:

> SoloForge: Mastering Project Autonomy

## البنية

- مجلد `alx-project-0x02/` يحتوي على مشروع Next.js كامل.
- يمكن تشغيل المشروع بالأوامر:

```bash
cd alx-project-0x02
npm install
npm run dev -- -p 3000
```

ثم زر الرابط: <http://localhost:3000>
